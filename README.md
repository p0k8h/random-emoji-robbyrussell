## How to install?

```
1. clone the repo

2. cd random-emoji-robbyrussell

3. sh install.sh random-emoji-robbyrussell

4. vi ~/.zshrc

5. ZSH_THEME="random-emoji-robbyrussell"

6. Save and reload the terminal and see changes.
```
